# Supabase Database Setup

Untuk mengaktifkan fitur real-time tracking proposal, Anda perlu menjalankan script SQL berikut di SQL Editor Supabase Anda.

## 1. Tabel `riwayat_download`

```sql
-- Buat tabel riwayat_download
CREATE TABLE IF NOT EXISTS public.riwayat_download (
    id BIGINT GENERATED BY DEFAULT AS IDENTITY PRIMARY KEY,
    nomor_surat TEXT NOT NULL,
    tujuan_surat TEXT NOT NULL,
    pemohon TEXT NOT NULL,
    tanggal_surat DATE NOT NULL DEFAULT CURRENT_DATE,
    no_urut INTEGER NOT NULL,
    link_download TEXT,
    created_at TIMESTAMPTZ DEFAULT NOW()
);

-- Aktifkan Realtime untuk tabel ini
ALTER PUBLICATION supabase_realtime ADD TABLE public.riwayat_download;

-- Berikan izin akses publik (jika menggunakan anon key tanpa RLS ketat)
-- Catatan: Untuk produksi, disarankan menggunakan kebijakan RLS yang lebih aman.
ALTER TABLE public.riwayat_download ENABLE ROW LEVEL SECURITY;
CREATE POLICY "Akses Publik Seluruhnya" ON public.riwayat_download FOR ALL USING (true);
```

## 2. Fungsi RPC `get_next_nomor_surat`

Fungsi ini digunakan untuk menghitung nomor surat berikutnya secara otomatis di sisi database.

```sql
CREATE OR REPLACE FUNCTION get_next_nomor_surat()
RETURNS TEXT AS $$
DECLARE
    next_no INTEGER;
    current_month_roman TEXT;
    current_year_short TEXT;
    months_roman TEXT[] := ARRAY['I', 'II', 'III', 'IV', 'V', 'VI', 'VII', 'VIII', 'IX', 'X', 'XI', 'XII'];
BEGIN
    -- Mengambil no_urut terakhir, mulai dari 13 jika tabel kosong
    SELECT COALESCE(MAX(no_urut), 12) + 1 INTO next_no FROM public.riwayat_download;
    
    current_month_roman := months_roman[extract(month from now())];
    current_year_short := to_char(now(), 'YY');
    
    -- Format: 13/VI-‘26/MJ-BA/PPSGGR30-1
    RETURN next_no || '/' || current_month_roman || '-‘' || current_year_short || '/MJ-BA/PPSGGR30-1';
END;
$$ LANGUAGE plpgsql;
```

## 3. Fungsi RPC `create_proposal`

Fungsi ini menyisipkan data proposal baru dan mengembalikan baris data yang baru dibuat.

```sql
CREATE OR REPLACE FUNCTION create_proposal(pemohon_name TEXT, tujuan TEXT)
RETURNS SETOF public.riwayat_download AS $$
DECLARE
    next_no INTEGER;
    new_nomor_surat TEXT;
BEGIN
    -- Hitung no_urut berikutnya
    SELECT COALESCE(MAX(no_urut), 12) + 1 INTO next_no FROM public.riwayat_download;
    
    -- Generate nomor surat
    new_nomor_surat := get_next_nomor_surat();
    
    RETURN QUERY
    INSERT INTO public.riwayat_download (nomor_surat, tujuan_surat, pemohon, no_urut, tanggal_surat)
    VALUES (new_nomor_surat, tujuan, pemohon_name, next_no, CURRENT_DATE)
    RETURNING *;
END;
$$ LANGUAGE plpgsql;
```
